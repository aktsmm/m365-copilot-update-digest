### Microsoft Copilot (Microsoft 365): Agent Builder から Agent Store にエージェントを送信する
- ソース: Microsoft 365 Roadmap
- 領域: Microsoft 365 Copilot
- 公開日: 2026/05/30
- なぜ重要か: 提供開始済みで利用可否の確認に直結するため / 管理やガバナンスへの影響が大きいため
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557173
お客様は、M365 管理センターで管理者がレビューおよび承認した後、Agent Builder で構築されたカスタム Copilot エージェントを「組織によって構築」セクションで Agent Store に送信できます。この管理されたフローにより、管理者は送信されたエージェントを確認、承認、公開できるため、他のユーザーがエージェントを検出して使用できるようになります。

- 領域: Microsoft Edge

Copilot が提案するアクションや厳選された作業コンテンツを使用して 1 つの検索ボックスから Web を検索、チャット、探索できるため、新しく更新された Copilot の新しいタブ ページでユーザーが作業を迅速に完了できるようになります。ユーザーは、パーソナライズされたニュースやアクティビティを表示することもできます。
