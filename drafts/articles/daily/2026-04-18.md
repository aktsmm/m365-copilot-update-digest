### Microsoft Copilot (Microsoft 365): ビジュアルでよりスマートになりました: Copilot Chat は埋め込み画像を活用して、より豊富で正確な回答を提供し...
- ソース: Microsoft 365 Roadmap
- 領域: Microsoft 365 Copilot
- 公開日: 2026/04/18
- なぜ重要か: 今後の展開計画や事前準備に影響するため / 管理やガバナンスへの影響が大きいため
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560540
Copilot は、Word 文書 (.docx)、PowerPoint プレゼンテーション (.pptx)、PDF などのファイルに埋め込まれた画像を使用して応答を解釈し、グラウンディングするように強化されました。この機能により、Copilot はチャート、図、スクリーンショットなどの視覚要素から洞察を抽出できるようになり、答えがテキストだけではないことが保証されます。
### Outlook: Copilot は Outlook の検索結果に簡潔な概要を提供するようになりました
- ソース: Microsoft 365 Roadmap
- 領域: Microsoft 365 Copilot
- 公開日: 2026/04/18
- なぜ重要か: 今後の展開計画や事前準備に影響するため / 管理やガバナンスへの影響が大きいため
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559989
Outlook の検索結果の上部に Copilot の簡潔な AI 概要が表示されるため、クエリに対する回答がすぐに得られます。ユーザーは、Copilot 概要カードから直接 Copilot チャットを開くことで、より深く掘り下げることができます。
