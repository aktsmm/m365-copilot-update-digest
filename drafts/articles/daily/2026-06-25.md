### Copilot ノートブック から Word 文書生成に対応
- ソース: Microsoft 365 Roadmap
- 領域: Microsoft 365 Copilot
- 公開日: 2026/06/25
- なぜ重要か: 提供開始済みで利用可否の確認に直結するため / 管理やガバナンスへの影響が大きいため
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558934
Copilot ノートブックに収集したコンテンツや参照情報を使って Word 文書を生成できます。Copilot がノートブックのコンテキストに基づいた構造化ドキュメントを下書きし、Word で開いて編集できます。アイデアや素材から完成度の高い文書を効率的に作成できます。
### Microsoft Teams の会議・チャット機能を更新
- ソース: Microsoft 365 Roadmap
- 領域: Microsoft Teams
- 公開日: 2026/06/25
- なぜ重要か: 今後の展開計画や事前準備に影響するため / 管理やガバナンスへの影響が大きいため
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558341
Teams 会議中に参加者から出た未回答の質問を Facilitator が自動検出し、Web 検索で回答を提案するようになりました。参加者が質問しても誰も答えない場合、Facilitator が回答を検索するか確認します。「はい」を選択するとエージェントが回答を返します。
### Microsoft Purview の会議・チャット機能を更新
- ソース: Microsoft 365 Roadmap
- 領域: Microsoft Purview
- 公開日: 2026/06/25

Microsoft Purview の DLP を Microsoft 365 Copilot と Copilot Chat に拡張し、外部メールに起因するリスクに対処します。組織外の送信者からのメールを Copilot が処理しないようリアルタイムで制御し、データ漏えいのリスクを低減します。
