### Microsoft 365 Copilot GCC general availability [Windows, Web]

- ソース: Microsoft 365 Copilot release notes
- 公開日: 2024/12/17
- リリース段階: GA
- 役割タグ: 作成者向け

GCC (Government Community Cloud) 環境での作業のための AI アシスタント。大規模言語モデル (LLM) の力を仕事の内容やコンテキストと組み合わせて、下書きと書き直し、要約と整理、見逃した部分の補足、質問への答えの取得に役立ちます。

### Microsoft 365 Copilot のセキュリティ・管理・分析機能を強化

- 原題: Generate keyword query language from natural language prompt in eDiscovery with Copilot for Security [Web]
- ソース: Microsoft 365 Copilot release notes
- 公開日: 2024/12/17
- リリース段階: Update
- 役割タグ: 管理者向け / 作成者向け
- URL: https://learn.microsoft.com/en-us/purview/ediscovery-natural-language-query

Copilot for Security は電子情報開示に組み込まれており、ユーザーが自然言語で検索プロンプトを提供できるようにし、キーワード クエリ言語に翻訳して電子情報開示検索の開始を促進します。もっと詳しく知る。
