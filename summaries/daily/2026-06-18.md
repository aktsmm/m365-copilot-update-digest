## 全件

### PowerPoint エージェント

- 原題: Microsoft Copilot (Microsoft 365): PowerPoint Agent
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/18
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=543421

PowerPoint Agent は、強力なストーリーテリングと視覚的構造を備えた洗練されたプレゼンテーションを作成するのに役立ちます。調査、書式設定、レイアウトが効率化されるため、メッセージに集中し続けることができます。マルチターン チャットを使用してコンテンツを調整したり、PowerPoint でデッキを開いて完全な編集やコラボレーションを行ったりできます。エグゼクティブ・プレジデントに最適です。

### 共有メールボックスと代理メールボックスで Copilot Chat が利用可能

- 原題: Outlook: Copilot Chat available in shared and delegate mailboxes
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/18
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=554936

読み取りと管理 (フル アクセス)、またはメール フォルダー レベルのアクセス許可を持つ共有メールボックス ユーザーと代理人は、新しい Outlook の共有メールボックスと代理人メールボックスから直接 Copilot Chat にアクセスできるようになります。

- 原題: Planner: Available in Outlook
- ソース: Microsoft 365 Roadmap

- リリース段階: In development
- 役割タグ: 作成者向け

Outlook の Planner は、すべてのタスク、計画、新しいプロジェクト マネージャー エージェントを直接受信トレイに取り込むため、すでに作業している場所で作業を管理できます。

### データ セキュリティ体制管理 - DSPM のデータ セキュリティ体制エージェント

- 原題: Microsoft Purview: Data Security Posture Management-Data Security Posture Agent in DSPM
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/18
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=542790

Data Security Posture Agent は、組織内の動的なデータとリスクの状況を常に把握するために積極的に取り組むデータ セキュリティ管理者の能力を拡張するように設計されています。その主な仕事は、データ資産全体にわたる機密データの検出を支援することです。このエージェントは、ドキュメントを分析および検索するように設計されています。

### カスタム スキルを使用して SharePoint で AI を拡張する

- 原題: SharePoint: Extending AI in SharePoint using custom skills
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/18
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559800

スキルにより SharePoint AI エージェントの拡張が可能になり、固有のビジネス ニーズに適応するカスタマイズ可能なソリューションが可能になります。これらは、独自のビジネス プロセス向けのすぐに使用できる機能のギャップに対処し、再利用可能な複数ステップのワークフローを可能にします。

- 公開日: 2026/06/18
- リリース段階: GA
- 役割タグ: 作成者向け
- URL: https://techcommunity.microsoft.com/t5/copilot-studio-blog/meet-the-new-copilot-studio-rebuilt-for-more-complex-multi-step/ba-p/4526488

私たちは常にお客様からのフィードバックを聞いています。あなたは、自然に連携して機能する、より有能なエージェントとワークフローを構築したいと考えています。複数のステップからなるタスクを、途中で中断することなく確実に処理できるエージェントが必要です。エージェントを構築するには、より直感的な方法が必要です。

### データ損失防止 – Microsoft Foundry アプリとエージェントのプロンプト用インライン DLP

- 原題: Microsoft Purview: Data Loss Prevention – Inline DLP for Prompts for Microsoft Foundry apps and agents
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/18
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558565

Microsoft Foundry で Purview を有効にすると、Purview 管理者は、Foundry で構築されたアプリやエージェントにプロンプトを表示するインライン DLP ポリシーを設定して、機密データのデータ損失を確実に防ぐことができます。
