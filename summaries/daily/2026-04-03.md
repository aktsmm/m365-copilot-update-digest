## 全件

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Outlook: Prepare for your meeting with Copilot chat in Outlook mobile
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/03
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=516568

Outlook mobile users can now prepare for their meetings using Copilot chat in a single tap. This feature requires a Microsoft 365 Copilot license.

### コード インタープリタ機能の拡張

- 原題: Microsoft Copilot (Microsoft 365): Files found via Chat search can be now analyzed by code interpreter
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/03
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=519570

Code interpreter runs on files you upload in your agent or that the user explicitly attaches in the interaction. Files that Copilot Chat finds via enterprise search can be now analyzed by code interpreter without the need to explicitly link or upload them.

### 埋め込み画像を使った応答精度向上に対応

- 原題: Microsoft Copilot (Microsoft 365): Now smarter with visuals: Declarative Agents leverage embedded images for richer, more accurate answers
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/03
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=531758

Declarative Agents have been enhanced to interpret and ground responses using images embedded in files such as Word documents (.docx), PowerPoint presentations (.pptx), and PDFs. This capability allows the agent to extract insights from visual elements like charts, diagrams, and...

### Copilot Notebooks の要約・インサイトを強化

- 原題: Microsoft 365 app: Overview Page in Copilot Notebooks
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/03
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=512430

Users will find Copilot-powered Notebook summaries and insights as a landing page within their Copilot Notebook.
