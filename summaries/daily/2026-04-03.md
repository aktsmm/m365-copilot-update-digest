## 全件

### Microsoft 365 Copilot の連携機能を拡張

- 原題: Microsoft Copilot (Microsoft 365): Admins can create and deploy custom Model Context Protocol (MCP) connectors across Microsoft 365.
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/03
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=501585

管理者は、Microsoft 365 管理センターでカスタムのモデル コンテキスト プロトコル (MCP) コネクタを作成し、Microsoft 365 全体に展開できます。 これにより、管理者は、これらの MCP ベースのコネクタを介して Microsoft 365 Copilot で組織のカスタム データを表示できるようになります。

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Copilot (Microsoft 365): Voice chat in Microsoft 365 Copilot
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/03
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=481138

Microsoft 365 Copilot の音声は、デスクトップやモバイル デバイスに自然なハンズフリーの会話エクスペリエンスをもたらします。これにより、同僚のように Copilot と会話できるようになり、より迅速で直感的な作業方法が実現します。 M365 Copilot に直接話しかけて、情報を検索したり、会議の合間に情報を共有したり、ブレインストーミングを行ったりできます。

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Teams: Book future meetings directly from Teams panels
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/03
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557168

デバイス上のカレンダーを参照し、翌日の午前 0 時までの空いている時間帯を選択することで、Teams パネルから今後の会議を予約できるようになりました。予約時にゲストを追加すると、アドホックなスケジュール設定と調整が合理化されます。 Teams Rooms Pro および共有デバイス ライセンスで利用できます。

### Microsoft 365 Copilot のセキュリティ・管理・分析機能を強化

- 原題: Microsoft Purview: Role Group changes in Purview
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/03
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=551147

新しい Microsoft Purview RBAC ロール (Purview Agent Deployment) を導入し、それを Purview 全体のアナリストや管理者が使用するさまざまな既存の組み込みロール グループに追加します。この変更により、組み込みのロール アナリスト グループを使用するユーザーは、追加のロールを必要とせずに Purview に Security Copilot Agent を展開できるようになります。
