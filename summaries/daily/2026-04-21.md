### Microsoft Edge: Microsoft Edge は、M365 Copilot 検索で検索するときに、より関連性の高い作業結果を得るために、仕事関連の閲覧履歴を Copilot...

- 原題: Microsoft Edge: Microsoft Edge can now send work related browsing history to Copilot for more relevant work results when searching on M365 Copilot search
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/21
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=537283

Microsoft Edge は、IT 管理者が選択した仕事関連の閲覧履歴を Copilot に送信して、検索の関連性を向上できるようになりました。ユーザーが Copilot 検索で何かを検索すると、Copilot は、関連する場合はブラウザ履歴の結果も表示します。
