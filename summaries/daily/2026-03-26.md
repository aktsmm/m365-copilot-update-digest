## 全件

### PowerPoint で画像の即時編集に対応

- 原題: Microsoft Copilot (Microsoft 365): Quickly edit an image in PowerPoint
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/26
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=551187

Copilot を使用して PowerPoint で画像を直接編集できます。アプリを切り替えたり、フローを中断したりする必要はありません。ピッチデッキのビジュアルを調整したり、マーケティング資産を強化したり、単にスライドをポップにしたりする場合でも、画像エディターを使用すると迅速かつシームレスに作業を行うことができます。解像度を向上させたり、解像度を削除したりするなどの編集を行うことができます。

### Microsoft 365 Copilot の連携機能を拡張

- 原題: Microsoft Copilot (Microsoft 365): Federated Copilot Connectors
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/26
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=501120

Microsoft は、Microsoft 365 Copilot のフェデレーション Copilot コネクタの一般提供 (GA) を発表します。 Federated Copilot コネクタを使用すると、ユーザーは Copilot を一般的なサードパーティ ソースに安全に接続し、Model Context Protocol (MCP) を使用してリアルタイムでデータを取得できます。これらのコネクタは、データを保存したり、保存したりすることはできません。
