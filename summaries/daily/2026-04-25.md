### Microsoft Viva: 公開レポート受信者の拡張

- 原題: Microsoft Viva: Expanded published report recipients
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/25
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558257

これにより、アナリストは Viva Insights Power BI レポートを組織内の誰にでも公開できるようになります。現在、アナリストは Copilot ダッシュボードへのアクセス権を持つユーザーにのみそれらを公開できます。

### Microsoft Copilot (Microsoft 365): Outlook の Copilot Chat は、受信トレイ、カレンダー、エンタープライズ データの推論に拡張されます。

- 原題: Microsoft Copilot (Microsoft 365): Copilot Chat in Outlook expands to reason over inbox, calendar, and enterprise data
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/25
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=531910

Outlook の Copilot Chat は、Microsoft 365 Copilot Chat ユーザー (Microsoft 365 Copilot ライセンスなし) のユーザーの受信トレイ全体、予定表、その他の企業データに対する推論をサポートするように拡張されています。

- リリース段階: Launched

### Microsoft Copilot (Microsoft 365): 「Hey Copilot」と言うと、Windows 上の Microsoft 365 Copilot アプリとの音声会話...

- 原題: Microsoft Copilot (Microsoft 365): Say "Hey Copilot" to activate voice conversations with the Microsoft 365 Copilot app on Windows
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/25
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=497848

Windows 上の Microsoft 365 Copilot アプリを使用して、「Hey Copilot」と言うと、Copilot とのリアルタイム音声会話がアクティブになります。ウェイク ワードにより、別のアプリやウィンドウに移動することなく、コンテンツの下書き、データ分析、重要な洞察の取得がスムーズに行えるようになります。エンタープライズ向けに構築されており、組織を尊重します...

### Microsoft Copilot (Microsoft 365): Microsoft 365 Copilot アプリのブランド フッター

- 原題: Microsoft Copilot (Microsoft 365): Branded footer in the Microsoft 365 Copilot app
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/25
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=555852

この更新プログラムでは、Microsoft 365 Copilot アプリに共同ブランド エクスペリエンスが導入され、組織が管理者によって承認された視覚的な識別子をチャット画面内に直接表示できるようになります。具体的には、このエクスペリエンスにより、管理者は Microsoft 管理ツールですでに管理されている組織でサポートされているロゴを活用できるようになります。

### Microsoft Copilot (Microsoft 365): Copilot は、文書構造を使用して長いファイルのナビゲーションを改善し、より正確な回答とより明確な引用を可能にしまし...

- 原題: Microsoft Copilot (Microsoft 365): Copilot improved long files navigation using document structure, enabling more accurate answers and clearer citations.
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/25
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559614

Copilot が文書構造をより認識できるようにすることで、Copilot が長く複雑なファイルを読み取る方法を改善しました。質問すると、Copilot は回答する前にドキュメントの最も関連性の高い部分 (特定のセクション/ページなど) に移動できるようになりました。これにより、詳細の見逃しが減り、明確な説明で信頼性が向上します。
