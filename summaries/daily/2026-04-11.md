## 全件

### Teams にエージェントを共有可能に

- 原題: Microsoft Copilot (Microsoft 365): Share agents to Teams
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557947

Teams チームにエージェントを共有できるようになります。エージェント共有ダイアログでチームを検索し、メイン チャンネルへ通知することでメンバーへの周知とインストールを簡単に行えます。

### PowerPoint Live でスライド選択の説明機能を追加

- 原題: Microsoft Copilot (Microsoft 365): Explain slide selection during PowerPoint Live
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557256

PowerPoint Live の会議体験を強化します。Copilot を活用して出席者がスライドのテキストを選択し、その内容の説明を取得できるようになります。

### ナレッジ エージェント機能を追加

- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け

### PowerPoint で画像生成・編集モデルを選択可能に

- 原題: PowerPoint: Create and Edit images with the model of your choice

- リリース段階: In development

- ソース: Microsoft 365 Roadmap

### Purview DLP で Copilot の機密データ保護を強化

- 原題: Microsoft Purview: Data Loss Prevention to prevent Microsoft 365 Copilot processing content with sensitivity labels supports all storage locations
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: Rolling out
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557255

Word、Excel、PowerPoint の機密ラベル付きファイルを Copilot が処理しないよう制限する Microsoft Purview DLP ポリシーが、ファイルの保存場所に関わらず適用されるようになります。

### ファイル プレビューで Copilot アクション提案を表示

- 原題: OneDrive: Discover Copilot actions in File Preview
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: Rolling out
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559481

OneDrive や SharePoint でファイルをプレビューした際に、Copilot の利用可能な操作が即座に表示されます。すぐに使えるプロンプトが Copilot ボタンの横に表示され、文書の要約や FAQ 生成などをプロンプトを書かずに実行できます。
