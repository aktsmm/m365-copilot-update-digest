## 全件

### Teams にエージェントを共有可能に

- 原題: Microsoft Copilot (Microsoft 365): Share agents to Teams
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557947

ユーザーが作成したエージェントを Microsoft Teams チームに共有できます。エージェント共有ダイアログからチームを検索・選択でき、メインチャンネルへの通知送信も可能です。

### Microsoft Copilot (Microsoft 365): PowerPoint Live 中のスライド選択について説明する

- 原題: Microsoft Copilot (Microsoft 365): Explain slide selection during PowerPoint Live
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557256

PowerPoint Live 会議中に参加者がスライドのテキストを選択し、Copilot から内容の説明を受けられます。Copilot を活用して PowerPoint Live の会議体験を向上させる機能です。

### Microsoft Copilot (Microsoft 365): ナレッジ エージェント

- 原題: Microsoft Copilot (Microsoft 365): Knowledge Agent
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=501451

SharePoint のコンテンツ管理を簡素化し、メタデータの自動化・古いコンテンツの修正・高速なページ作成・コンプライアンス向上などにより、Copilot がより正確で根拠のある回答を提供できるようにします。

### PowerPoint で画像生成・編集モデルを選択可能に

- 原題: PowerPoint: Create and Edit images with the model of your choice
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559479

PowerPoint の Copilot で画像を作成・編集する際に、OpenAI の GPT Image・Flux・Auto (スマート動的選択) から好みのモデルを選択できます。新規プレゼンテーションやページの生成時にも対応しています。

### Purview DLP で Copilot の機密データ保護を強化

- 原題: Microsoft Purview: Data Loss Prevention to prevent Microsoft 365 Copilot processing content with sensitivity labels supports all storage locations
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/11
- リリース段階: Rolling out
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557255

Microsoft Purview DLP ポリシーにより、Word・Excel・PowerPoint 内の機密ファイルへの Microsoft 365 Copilot の処理制限が、ファイルの保存場所に関わらず適用されるようになります。
