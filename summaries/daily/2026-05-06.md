### Federated Copilot コネクタ - Microsoft 365 Copilot 内にリアルタイムのエンタープライズ データを導入

- 原題: Federated Copilot connectors - bringing real-time enterprise data within Microsoft 365 Copilot
- ソース: Microsoft 365 Copilot Blog
- 公開日: 2026/05/06
- リリース段階: GA
- 役割タグ: 管理者向け / 作成者向け
- URL: https://techcommunity.microsoft.com/t5/microsoft-365-copilot-blog/federated-copilot-connectors-bringing-real-time-enterprise-data/ba-p/4515993

Microsoft 365 Copilot は、Work IQ をさらに強化し、数百の SaaS システムにわたるエンタープライズ ナレッジを Microsoft 365 テナント内の単一の一貫したエンタープライズ ナレッジ ベースに統合できる同期機能を備えた Copilot コネクタを活用します。この基盤により、Copilot が可能になります。
