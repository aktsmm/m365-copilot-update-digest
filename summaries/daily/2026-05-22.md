### Microsoft Copilot (Microsoft 365): Copilot チャットで Word、Excel、PowerPoint ファイルを開く

- 原題: Microsoft Copilot (Microsoft 365): Open Word, Excel, and PowerPoint Files in Copilot Chat
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/22
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=548641

現在、Copilot で引用されたコンテンツをクリックすると、Copilot の外部のファイルが開きます。この新機能を使用すると、Copilot Chat でコンテンツを直接開くことができ、チャットと同時にドキュメントを表示することがより合理化されます。この変更は、Microsoft 365 Copilot モバイル アプリ、または Edge で Copilot、Copilot を使用している場合には利用できません。

### Microsoft Teams: Microsoft Teams の Microsoft 365 管理エージェントを使用して SMB 管理タスクをより高速に実行します

- 原題: Microsoft Teams: Run SMB admin tasks faster with Microsoft 365 Admin Agent in Microsoft Teams

- 公開日: 2026/05/22
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558255

SMB 管理者は、Microsoft Teams の Microsoft 365 管理エージェントを使用して、一般的な管理タスクを完了し、重要な設定アクションに関するガイダンスを得ることができます。管理エージェントは、管理者に代わってユーザーを追加し、ライセンスを割り当てることができるほか、組織のセットアップ、セキュリティ設定、パスワードなどの主要な SMB 関連トピックに関するガイダンスを提供します。
