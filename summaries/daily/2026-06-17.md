### 開いている YouTube ビデオで Edge for Business の Microsoft 365 Copilot チャットを実行します。

- 原題: Microsoft Edge: Ground Microsoft 365 Copilot Chat in Edge for Business in your open YouTube videos
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/17
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=499424

Edge の Copilot Chat で、開いている YouTube ビデオに関する関連する回答と洞察を取得します。

### 会議録画の機密ラベルの継承

- 原題: Microsoft Teams: Sensitivity label inheritance for meeting recordings
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/17
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557178

会議の録画は、会議の機密ラベルを自動的に継承するようになりました。機密ラベル ポリシーでラベルの継承が有効になっている場合、ラベル付き会議は同じラベルを MP4 録音に適用し、アクセス制御、データ処理ルール、トランスクリプトに基づくCopilotまたはエージェントの応答が一貫して保証されるようにします。

### ページ コンテンツへの Microsoft 365 Copilot Chat アクセスが、複数の開いているタブにわたって機能するように間もなく拡張されます。

- 原題: Microsoft Edge: Microsoft 365 Copilot Chat access to page content will soon extend to work across multiple open tabs
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/17
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=496364

Edge の Microsoft 365 Copilot Chat は、ユーザーが開いているウィンドウに関する状況に応じた質問に間もなく回答できるようになります。これにより、開いているページにのみアクセスできる既存の機能が拡張されます。既存のユーザー設定とポリシー、および DLP に関する考慮事項はすべて引き続き尊重されます。

### 通訳者用のテナントレベルのダッシュボード

- 原題: Microsoft Teams: Tenant-level dashboard for Interpreter
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/17
- リリース段階: In development
- 役割タグ: 管理者向け

管理者は、Teams 管理センターに専用のダッシュボードを使用して、組織全体のインタープリターの使用状況を確認できるようになりました。導入と使用の傾向を 1 か所で追跡して、Interpreter がどのように使用されているかをよりよく理解できます。
