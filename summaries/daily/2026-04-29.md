### Microsoft Teams: Microsoft Teams の Microsoft 365 管理エージェントを使用して SMB 管理タスクをより高速に実行します

- 原題: Microsoft Teams: Run SMB admin tasks faster with Microsoft 365 Admin Agent in Microsoft Teams
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/29
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558255

SMB 管理者は、Microsoft Teams の Microsoft 365 管理エージェントを使用して、一般的な管理タスクを完了し、重要な設定アクションに関するガイダンスを得ることができます。管理エージェントは、管理者に代わってユーザーを追加し、ライセンスを割り当てることができるほか、組織のセットアップ、セキュリティ設定、パスワードなどの主要な SMB 関連トピックに関するガイダンスを提供します。

### Microsoft Copilot (Microsoft 365): Copilot の応答に含まれる追加の会議結果

- 原題: Microsoft Copilot (Microsoft 365): Additional meeting results in Copilot responses
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/29
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559111

現在、Copilot は最大 25 件の会議結果を表示できるため、大きなカレンダーでの表示が制限される可能性があります。この制限を拡大することで、より多くの関連性のある会議を一度に表示できるようになり、必要な完全なコンテキストを簡単に見つけて確認し、それに基づいて行動できるようになります。
