## 全件

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Outlook: Prepare for your meeting with Copilot chat in Outlook mobile
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/04
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=516568

Outlook モバイル ユーザーは、Copilot チャットを使用して 1 回のタップで会議の準備をできるようになりました。この機能には Microsoft 365 Copilot ライセンスが必要です。

### コード インタープリタ機能の拡張

- 原題: Microsoft Copilot (Microsoft 365): Files found via Chat search can be now analyzed by code interpreter
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/04
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=519570

コード インタープリターは、エージェントでアップロードしたファイル、またはユーザーがインタラクションで明示的に添付したファイル上で実行されます。 Copilot Chat がエンタープライズ検索で見つけたファイルは、明示的にリンクしたりアップロードしたりしなくても、コード インタープリターで分析できるようになりました。

### 埋め込み画像を使った応答精度向上に対応

- 原題: Microsoft Copilot (Microsoft 365): Now smarter with visuals: Declarative Agents leverage embedded images for richer, more accurate answers
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/04
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=531758

宣言型エージェントは、Word 文書 (.docx)、PowerPoint プレゼンテーション (.pptx)、PDF などのファイルに埋め込まれた画像を使用して応答を解釈および根拠付けるように強化されました。この機能により、エージェントはチャート、図、スクリーンショットなどの視覚要素から洞察を抽出し、回答が確実に得られるようにすることができます。

### Copilot Notebooks の要約・インサイトを強化

- 原題: Microsoft 365 app: Overview Page in Copilot Notebooks
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/04
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け

ユーザーは、Copilot を利用した Notebook の概要と洞察を、Copilot Notebook 内のランディング ページとして見つけることができます。
