## 全件

### Microsoft Viva: Copilot ダッシュボード - M365 Copilot Chat の洞察への幅広いアクセス

- 原題: Microsoft Viva: Copilot Dashboard - Broader access to M365 Copilot Chat insights
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/16
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=501581

Microsoft Viva は、1 ～ 49 個の M365 Copilot ライセンスを持つテナントを有効にすることで、Microsoft Copilot ダッシュボードの M365 Copilot Chat Insights へのアクセスを拡張しています。これにより、ライセンスの少ないテナントを含め、より多くのテナントが導入傾向を追跡できるようになります。インサイトには、経時的な使用状況、グループ レベルの導入、保持、アプリ レベルのブラウズが含まれます。

### Outlook: 従来の Windows 版 Outlook で Copilot を使用した会議の準備をする

- 原題: Outlook: Prepare for meetings with Copilot in classic Outlook for Windows
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/16
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=542186

立て続けに会議に参加する人が非常に多いため、事前に読んだ内容、実行事項、さらには各会議の内容を把握し続けるのは非常に困難です。 Copilot は、リアルタイムの洞察をもたらし、関連するコンテキスト、タスク、ドキュメント、その他のリソースを要約することで、数分以内に会議の準備を迅速に行います。できます...

### Outlook: 従来の Windows 版 Outlook での Copilot チャットを使用したスケジュール設定

- 原題: Outlook: Scheduling with Copilot chat in classic Outlook for Windows
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/16
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=542185

ユーザーはチャットから直接、または UI の [Copilot でスケジュール] をクリックして会議をスケジュールできるようになりました。 Copilot に同僚との会議のスケジュールを設定するよう依頼するだけで、利用可能な時間を検索し、部屋を予約し、議題を草案し、招待状を送信するなど、すべてチャット エクスペリエンス内で実行されます。この機能は、Microsoft を使用しているユーザーが利用できます。

- 原題: Microsoft Teams: Ad-hoc room reservation from Teams Rooms on Windows console

- リリース段階: In development

### Microsoft Teams: 知識を保持するための AI 生成の会議アーカイブ

- 原題: Microsoft Teams: AI-generated meeting archive for knowledge retention
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/16
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558933

管理者は、Copilot (およびファシリテーター) がトランスクリプトの保存期間を超えて会議からの重要な洞察を保持できるようにするテナント全体のポリシーを有効にすることができます。これにより、Copilot はトランスクリプトの有効期限が切れた後でも、過去の会議に関する質問に回答し続けることができます。管理者は、この AI によって生成された会議の継続時間を設定することもできます。
