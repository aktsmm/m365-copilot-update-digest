## 全件

### Copilot Notebook から PowerPoint プレゼンテーションを作成

- 原題: Microsoft Copilot (Microsoft 365): Create PowerPoint presentations from Copilot Notebooks
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/19
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558938

Copilot ノートブックを使用すると、ユーザーはノートブックに収集されたコンテンツや参考資料から PowerPoint プレゼンテーションを直接生成できます。 Copilot は、ノートブックの厳選されたコンテキストを使用して、ユーザーが PowerPoint で開いて調整できる構造化された編集可能なスライド デッキを作成し、調査、メモ、洞察をプロジェクトに変換するのに役立ちます。

### Copilot は、メールの下書き、編集、書式設定を行う際にチャットでコーチング フィードバックを提供します

- 原題: Outlook: Copilot provides coaching feedback in chat as you draft, edit and format emails
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/19
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559418

Copilot は、電子メールの下書きに関するコーチング フィードバックをチャット経由で提供し、Outlook のメッセージに提案を直接適用できるようになりました。ユーザーはどの提案を適用するかを選択することができ、Copilot によってメールが適切に更新されます。

### Microsoft 365 アプリ: Microsoft 365 Copilot アプリ (iOS) の軽量ノートブック エクスペリエンスが更新されました

- 原題: Microsoft 365 app: Updated light-weight Notebooks experience in the Microsoft 365 Copilot App (iOS)
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/19
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=566318

Microsoft 365 Copilot アプリ (iOS) の Copilot ノートブックを使用すると、ユーザーは関連するチャット、出力作成、参照を永続的な AI ワークスペースに整理できるようになりました。 Copilot は Notebook に蓄積されたコンテキストを使用して応答をグラウンディングするため、作業は毎回最初からやり直すのではなく、セッションをまたいで継続されます。

### Copilot ノートブックでのマルチモーダル キャプチャ (Windows)

- 原題: OneNote: Multimodal Capture in Copilot Notebooks (Windows)
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/19
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=566322

Windows の Capture を使用すると、会議、ブレーンストーミング セッション、日常業務から得た会話やアイデアを Copilot ノートブック に取り込むことができます。音声、画像、メモを 1 つのエクスペリエンスでキャプチャし、Copilot がそれらを重要な洞察、意思決定、アクション アイテムを含む構造化されたメモに変換して、Copilot ノートブックに保存します。

### Copilot Notebook から Word ドキュメントを作成する

- 原題: Microsoft Copilot (Microsoft 365): Create Word documents from Copilot Notebooks
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/19
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558934

Copilot ノートブックを使用すると、ユーザーはノートブックに収集されたコンテンツと参照を使用して Word ドキュメントを生成できます。 Copilot は、ノートブックのコンテキストに基づいて構造化された文書を作成します。文書は Word で開いて編集でき、ユーザーがアイデアやソース資料から、少ない労力ですぐに使える文書に移行できるようにします。
