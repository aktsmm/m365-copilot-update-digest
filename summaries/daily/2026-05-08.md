- 役割タグ: 作成者向け

### Microsoft Edge: Copilot の新しいタブ ページ

- 原題: Microsoft Edge: Copilot New Tab Page
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/08
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558256

Copilot が提案するアクションや厳選された作業コンテンツを使用して 1 つの検索ボックスから Web を検索、チャット、探索できるため、新しく更新された Copilot の新しいタブ ページでユーザーが作業を迅速に完了できるようになります。ユーザーは、パーソナライズされたニュースやアクティビティを表示することもできます。

### Microsoft Purview: データ損失防止 - Security Copilot による DLP ポリシー変更の洞察

- 原題: Microsoft Purview: Data Loss Prevention - DLP Policy Change Insights with Security Copilot
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/08
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559106

Microsoft Security Copilot を活用した AI を活用した機能を導入し、Microsoft Purview 内の各更新後に DLP ポリシーに加えられた変更についての自然言語説明を自動的に生成します。この機能は、ポリシー バージョンの差分 (更新前と更新後) を分析し、構造化された人間が読める形式のデータを生成します。

### データ損失防止 - Security Copilot による DLP ポリシー変更の洞察

- 原題: Data Loss Prevention- DLP Policy Change Insights with Security Copilot
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/08
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=472031

DLP のセキュリティのための Copilot は、DLP ポリシーを要約できます。この機能強化により、DLP 管理者はポリシーの適用範囲に関する洞察を得ることができます。場所、分類子と通知など。 DLP 管理者は、すべてのポリシーまたは選択したいくつかのポリシーについてこれらの分析情報を取得できます。プロンプトを使用してさらに深く理解できるようになります...
