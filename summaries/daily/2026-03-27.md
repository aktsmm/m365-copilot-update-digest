## 全件

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Teams: Ad-hoc room reservation from Teams Rooms on Windows console
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/27
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=548648

Windows コンソール上の Teams Rooms を使用すると、すぐに使用できる会議室をすぐに予約できるため、スケジュールの競合を回避し、中断のない自発的な会議を確保できます。 Teams Rooms Pro ライセンスが付与されたルームで利用できます。

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Teams: BYOD space peripheral health signals and reports in the Pro Management portal
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/27
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=493319

管理者は、プロ マネジメント ポータルのレポートを使用して、デバイスの持ち込み (BYOD) ルームやデスクで周辺機器を積極的に管理でき、デバイスに欠陥があるか、PC で検出できないか、紛失しているか、移動されているかどうかを示すようになりました。 Pro Management ポータルのレポートには、ルームの Teams 共有デバイス ライセンスが必要であり、デザイン用のレポートも必要です。

### Purview DLP で Copilot の機密データ保護を強化

- 原題: Microsoft Purview: Data Loss Prevention to prevent Microsoft 365 Copilot processing content with sensitivity labels supports all storage locations
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/27
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557255

Word、Excel、PowerPoint の機密ファイルの処理を Microsoft 365 Copilot で制限する Microsoft Purview データ損失防止ポリシーが、ファイルの保存場所に関係なく適用されるようになりました。

### ファイル プレビューで Copilot アクション提案を表示

- 原題: OneDrive: Discover Copilot actions in File Preview
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/27
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559481

OneDrive または SharePoint でファイルをプレビューした瞬間に Copilot で何ができるかをご覧ください。すぐに使用できるプロンプトが [Copilot] ボタンの横に表示され、プロンプトを 1 つも作成することなく、ドキュメントの要約、FAQ の生成などを行うことができます。
