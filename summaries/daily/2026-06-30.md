- リリース段階: Launched
- 役割タグ: 作成者向け

Microsoft 365 の Planner エージェントは、作業をより効率的に計画、整理、管理するのに役立ちます。私たちは、プロジェクトの追跡を簡素化するための主要なタスクと計画の管理機能から始めており、今後、追加の高度なプロジェクト管理機能を導入する予定です。 2026 年 5 月 21 日更新: 以前はプレビューでリリースされました...

- 公開日: 2026/06/30
- リリース段階: Launched

### Microsoft Teams の会議・チャット機能を更新

- 原題: Microsoft Teams: IntelliFrame people labels in Teams Rooms on Windows
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/30
- リリース段階: In development
- 役割タグ: 管理者向け

リモート会議の参加者は、室内に誰がいるのかを把握し、室内の参加者を識別し、インテリジェント カメラやクラウド上の音声と顔のプロファイルを使用してホバーオーバー連絡先カードを提供する新しい IntelliFrame 人物ラベルを使用して会話をより適切に追跡できます。 ... のライセンスが付与された Windows 上の Teams Rooms で利用できます。

### Copilot ノートブック の要約・インサイトを強化

- 原題: OneNote: Multimodal capture in Copilot Notebooks (Windows)
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/30
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=566322

Windows の Capture を使用すると、会議、ブレーンストーミング セッション、日常業務から得た会話やアイデアを Copilot ノートブック に取り込むことができます。音声、画像、メモを 1 つのエクスペリエンスでキャプチャし、Copilot がそれらを重要な洞察、意思決定、アクション アイテムを含む構造化されたメモに変換して、Copilot ノートブックに保存します。

### Microsoft 365 Copilot アプリの簡素化された設計

- 原題: Microsoft Copilot (Microsoft 365): Simplified design for the Microsoft 365 Copilot app
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/30
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け

Microsoft 365 Copilot Web およびデスクトップ アプリのデザインが刷新されたことで、ユーザー エクスペリエンスが簡素化され、Copilot との連携が促進されます。これには、チャット ホーム画面、チャット応答レイアウト、ナビゲーション ペインの更新が含まれます。
