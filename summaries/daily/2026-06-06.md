## 全件

### 権威サイト

- 原題: SharePoint: Authoritative Sites
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/06
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=561323

権威サイト機能を使用すると、管理者は特定の SharePoint サイトを公式で信頼できる情報源として指定できます。これらのサイトを権威あるサイトとして分類することで、企業ニュース、ポリシー、更新情報などの高品質で信頼できるコンテンツが、Copilot チャットおよび Copilot 検索サイト全体で優先されます。

### プランナー: グループベースの基本プランのプランナー エージェント

- 原題: Planner: Planner Agent in group-based basic plans
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/06
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=511820

Planner Agent は、基本プランを含むすべてのグループベースの Planner プランの M365 Copilot ライセンス ユーザーが利用できるようになります。

### Microsoft 365 Copilot モバイル アプリで音声メモをキャプチャします

- 原題: Microsoft Copilot (Microsoft 365): Capture voice notes in the Microsoft 365 Copilot mobile app
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/06
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=498640

Microsoft 365 Copilot ライセンスを使用すると、Copilot の音声メモを使用して、オフライン ディスカッションを構造化され、実用的で検索可能なコンテンツに変換できます。

### インサイト - 新しい Copilot メトリクスが利用可能

- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/06
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557981

Microsoft 365 Copilot アプリ、Edge、OneNote で実行されたアクションを含む一連の新しい Copilot 指標と、提案された返信、翻訳、コーチング、クリーン データなどを含む Outlook、Word、Excel、PowerPoint にわたる主要なユース ケースを提供します。これにより、お客様は主要な Copilot についてより深く理解できるようになります。

- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け

### Microsoft 365 Copilot の更新

- 原題: Microsoft Copilot (Microsoft 365): Reference an attached PowerPoint deck’s style when creating a new presentation with Agent Mode in PowerPoint
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/06
- リリース段階: In development
- 役割タグ: 作成者向け
