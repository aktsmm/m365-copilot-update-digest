## 全件

### Microsoft Teams の Microsoft 365 管理エージェントを使用して SMB 管理タスクをより高速に実行します

- 原題: Microsoft Teams: Run SMB admin tasks faster with Microsoft 365 Admin Agent in Microsoft Teams
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/24
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558255

SMB 管理者は、Microsoft Teams の Microsoft 365 管理エージェントを使用して、一般的な管理タスクを完了し、重要な設定アクションに関するガイダンスを得ることができます。管理エージェントは、管理者に代わってユーザーを追加し、ライセンスを割り当てることができるほか、組織のセットアップ、セキュリティ設定、パスワードなどの主要な SMB 関連トピックに関するガイダンスを提供します。

### Teams パネルのデジタル サイネージのサポート

- 原題: Microsoft Teams: Digital signage support for Teams panels
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/24
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558852

Microsoft Teams Rooms の前面ディスプレイのデジタル サイネージ サポートと同様に、アイドル状態の Teams パネル デバイスにデジタル サイネージを表示できるようになりました。サイネージのソースと設定は、Pro Management ポータルで構成できます。 Teams Rooms Pro または共有スペース ライセンスのスペースで利用できます。
