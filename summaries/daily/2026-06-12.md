### Microsoft 365: OneDrive 用の従量課金制ストレージ

- 原題: Microsoft 365: Pay-as-you-go storage for OneDrive
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/12
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=562352

追加の OneDrive ストレージ用の新しい従量課金制の消費量ベースのメーター。どの OneDrive アカウントがライセンスされたストレージ制限を超えて拡張できるかを制御し、選択したユーザーに必要に応じて追加のスペースを提供します。

### Work IQ API: 従量課金制の使用法

- 原題: Microsoft Copilot (Microsoft 365): Work IQ APIs: Pay‑As‑You‑Go usage
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/12
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559017

Work IQ エンドポイントへの従量課金制アクセスを有効にし、開発者が事前に割り当てられたライセンスを必要とせずにエージェントと機能を呼び出すことができ、リクエストごとに呼び出されたエージェントとモデルに基づいて使用量が請求されます。

### Teams で AI 生成の会議アーカイブに対応

- 原題: Microsoft Teams: AI-generated meeting archive for knowledge retention
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/12
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558933

管理者は、Copilot (およびファシリテーター) がトランスクリプトの保存期間を超えて会議からの重要な洞察を保持できるようにするテナント全体のポリシーを有効にすることができます。これにより、Copilot はトランスクリプトの有効期限が切れた後でも、過去の会議に関する質問に回答し続けることができます。管理者は、この AI によって生成された会議の継続時間を設定することもできます。

### 管理者によってブロックされたアプリとエージェントのリクエスト フローの改善

- 原題: Microsoft Teams: Improved request flows for apps and agents blocked by admins
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/12
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け

Teams では、明確でガイド付きのエンドツーエンドの要求エクスペリエンスを導入することで、Teams の管理者によってブロックされているアプリやエージェントへのアクセスをユーザーが要求する方法を改善しています。この取り組みにより、リクエスト プロセスが明確になり、リクエストに更新があった場合にユーザーに通知することで、エンド ユーザーの摩擦が軽減されます。充実に加えて...
