## 全件

### M365 Copilot Memory の個人最適化を強化

- 原題: Microsoft Copilot (Microsoft 365): Enhanced M365 Copilot Memory - Copilot is now personalized with work data
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/19
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=551195

Copilot は、M365 Graph データを使用して、より関連性の高い状況に応じた応答を提供するようになりました。更新された設定により、Copilot が記憶する内容と個人設定の表示と管理が容易になります。

### Microsoft 365 Copilot のセキュリティ・管理・分析機能を強化

- 原題: Microsoft Purview: Data Loss Prevention - Data Security Triage Agent summaries and categorizations for DLP alerts will be made available in DLP Alerts in Microsoft Defender XDR
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/19
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558860

DLP アラートのデータ セキュリティ トリアージ エージェントの概要と分類は、Microsoft Defender XDR の DLP アラートで利用できるようになります。
