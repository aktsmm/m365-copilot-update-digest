### Microsoft Teams: ファシリテーターが質問を検出して回答します

- 原題: Microsoft Teams: Facilitator detects and answers questions
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/23
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558341

ファシリテーターは、Teams 会議中に出された未解決の質問を自動的に検出し、Web 検索を使用して回答を提供するようになりました。参加者が質問 (「LLM とは何ですか?」など) をしても誰も答えない場合、ファシリテーターは答えを見つけるための提案を投稿します。参加者は「はい」を選択して、エージェントに撤回してもらうことができます。

### Microsoft Copilot (Microsoft 365): Microsoft Forms での調査エージェントと Copilot のチャット

- 原題: Microsoft Copilot (Microsoft 365): Surveys Agent and Copilot Chat in Microsoft Forms
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/23
- リリース段階: Rolling out
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=553136

Microsoft 365 Copilot Chat は、Forms 内から Surveys Agent を有効にするなど、Microsoft Forms に統合されました。 Surveys Agent は、フォームやクイズに対する改善の推奨と適用から、招待状の作成と送信の支援、アンケートの分析まで、フォームを支援するための特殊な機能を提供します。
