### Microsoft Copilot (Microsoft 365): 無限スクロール チャット履歴

- 原題: Microsoft Copilot (Microsoft 365): Infinite Scroll Chat History
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/24
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557348

無限スクロールにより、ユーザーはスクロール中に追加の会話を自動的に読み込むことで、古いチャット セッションを継続的に閲覧できます。これにより、よりスムーズで最新のチャット履歴エクスペリエンスが作成されます。GA 日: 2026 年 4 月

### Microsoft Purview: データ損失防止におけるデータ セキュリティ トリアージ エージェント - エージェントがトリアージしたアラートの推論トレースと信頼スコア

- 原題: Microsoft Purview: Data Security Triage Agent in Data Loss Prevention - Reasoning Trace and Confidence Score for agent-triaged alerts
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/24
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560597

この機能は、DLP のデータ セキュリティ トリアージ エージェントに信頼スコアリングと推論トレースの説明可能性を導入し、複数の顧客にわたって報告される重大な信頼ギャップに対処します。現在、エージェントが決定を下す理由や自信の度合いが透明性がないため、アナリストはマニュアルに頼らざるを得ません。
