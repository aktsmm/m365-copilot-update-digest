## 全件

### Power BI フィルタリング強化で Copilot インサイトを拡充

- 原題: Microsoft Viva: Unlocking Deeper Copilot Insights with Enhanced Power BI Filtering
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/27
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559995

グローバル管理者とインサイト管理者が、予約属性や組織のカスタム属性など、追加の属性を Power BI レポートのフィルターとして有効にできるようにします。 Power BI レポートにアクセスできるユーザーは、これらの新しく有効になったフィルターを使用して、ビジネス ニーズに合わせてレポートを調整できます。

- 原題: Outlook: Account Selector in Outlook Copilot Side Pane
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/27
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559991

ユーザーが複数の Copilot 対応アカウントを新しい Outlook に追加している場合、カレンダーの Copilot サイド ウィンドウにアカウント ドロップダウンが表示され、使用している Copilot 対応アカウントが明確になり、Outlook UI のアカウント コンテキストに依存せずにアカウントを切り替えることができます。

### Microsoft Purview のセキュリティ・管理・分析機能を強化

- 原題: Microsoft Purview: Azure AI Foundry integration with Microsoft Purview for AI
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/27
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=537271

AI Foundry での Purview の有効化により、Foundry 管理者はサブスクリプションで Microsoft Purview をアクティブ化できるようになります。有効にすると、すべてのアプリとエージェントからの AI インタラクション データが Purview に流れ込み、AI データの一元的なコンプライアンス、ガバナンス、態勢管理が可能になります。

- リリース段階: In development

### Microsoft Purview の会議・チャット機能を更新

- 原題: Microsoft Purview: Data Loss Prevention - Data Loss Prevention to safeguard sensitive information from external web search in Microsoft 356 Copilot and Copilot Chat
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/27
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=565870

Microsoft は、機密データを含む Web 検索を保護するために、Microsoft 365 Copilot および Copilot Chat 用の Microsoft Purview DLP を拡張しています。このリアルタイム制御は、Microsoft 365 Copilot とエージェントが外部の Web 検索に機密データを使用することを防ぎ、組織がデータ漏洩と過剰共有のリスクを軽減するのに役立ちます。

### Microsoft 365 Copilot の Copilot 機能を更新

- 原題: Microsoft Copilot (Microsoft 365): Copilot in PowerPoint for Government clouds
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/27
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=561322

PowerPoint の Copilot を使用すると、プレゼンテーション内で直接、自然な会話を通じてプレゼンテーションを作成、編集、調整できます。新しいプレゼンテーションを開始することも、既存のプレゼンテーションを基にして構築することもでき、書式設定や構造などを維持しながら、Copilot にスライドの生成、コンテンツの更新、レイアウトの改善、デザインの洗練を依頼できます。

- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け
