### Microsoft Copilot Studio: エージェント ノードを使用してワークフロー ステップとしてエージェントを呼び出す

- 原題: Microsoft Copilot Studio: Invoke agents as workflow steps with the agent node
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/15
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=562222

エージェント ノードをワークフローに追加すると、データの推論、ツールの呼び出し、応答の返しをすべて 1 つのステップ内で行うことができます。エージェント ノードを使用すると、推論をワークフローに直接埋め込むことができます。これは、手動の引き継ぎが減り、処理時間が短縮され、追加の開発を行わずに微妙な決定を処理できるワークフローを意味します。
