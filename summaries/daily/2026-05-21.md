### Microsoft Teams: Android 上の Teams Rooms での通訳エージェントのサポート

- 原題: Microsoft Teams: Interpreter agent support in Teams Rooms on Android
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/21
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=562665

通訳エージェントは Microsoft Teams 会議で翻訳者として機能し、参加者はリアルタイムの翻訳を利用して選択した言語で会議を聞くことができるため、異なる言語を話すことが効果的な理解とコラボレーションの障壁になることはありません。ライセンスを取得した Android 上の Teams Rooms で利用できるようになりました。

### Microsoft Teams: 今後の会議を Teams パネルから直接予約します

- 原題: Microsoft Teams: Book future meetings directly from Teams panels
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/21
- リリース段階: Rolling out
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557168

デバイス上のカレンダーを参照し、翌日の午前 0 時までの空いている時間帯を選択することで、Teams パネルから今後の会議を予約できるようになりました。予約時にゲストを追加すると、アドホックなスケジュール設定と調整が合理化されます。 Teams Rooms Pro および共有デバイス ライセンスで利用できます。

### Microsoft Copilot (Microsoft 365): Agent Builder から Agent Store にエージェントを送信する

- 原題: Microsoft Copilot (Microsoft 365): Submit agent to Agent Store from Agent Builder
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/21
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557173

お客様は、M365 管理センターで管理者がレビューおよび承認した後、Agent Builder で構築されたカスタム Copilot エージェントを「組織によって構築」セクションで Agent Store に送信できます。この管理されたフローにより、管理者は送信されたエージェントを確認、承認、公開できるため、他のユーザーがエージェントを検出して使用できるようになります。
