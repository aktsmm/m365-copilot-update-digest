### Microsoft 365 Copilot のセキュリティ・管理・分析機能を強化

- 原題: Microsoft Purview: Azure AI Foundry integration with Microsoft Purview for AI
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/18
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=537271

AI Foundry での Purview の有効化により、Foundry 管理者はサブスクリプションで Microsoft Purview をアクティブ化できるようになります。有効にすると、すべてのアプリとエージェントからの AI インタラクション データが Purview に流れ込み、AI データの一元的なコンプライアンス、ガバナンス、態勢管理が可能になります。

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Teams: Delete meeting generated content in recap
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/18
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557170

会議の主催者は、要約ページの新しい削除ボタンを使用して、録音、トランスクリプト、AI サマリー、メモなどの会議で生成されたコンテンツを簡単に削除できるようになりました。ミーティング内の共有ファイルは削除されません。
