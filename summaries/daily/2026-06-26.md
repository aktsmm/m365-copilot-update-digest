### Microsoft 365 Copilot のセキュリティ・管理・分析機能を強化

- 原題: Microsoft Purview compliance portal: Copilot Prompt Gallery in Data Security Posture Management
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/26
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=492618

データ セキュリティ態勢管理 (DSPM) の Copilot プロンプト ギャラリーは、ユーザーが DSPM で Security Copilot をより効果的に活用できるよう支援するプロンプトとプロンプトブックのキュレーション コレクションです。プロンプトを検索・探索し活用することでワークフローを効率化できる一元的なハブとして機能します。

- ソース: Microsoft 365 Roadmap

- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け

Microsoft 365 管理センターに Copilot コネクタの利用状況レポートが追加されます。組織はコネクタの詳細な利用状況を把握でき、コネクタ全体の活動状況やコネクタを活用した応答などのインサイトが得られます。

### Azure AI Search が Purview の秘密度ラベルとポリシーに対応

- 原題: Microsoft Purview: Information Protection-Azure AI Search honors Purview labels and policies
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/26
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=534610

Azure AI Search が Microsoft Purview の秘密度ラベルを取り込み、SharePoint・OneLake・Azure Blob・ADLS Gen2 などの組み込みインデクサーを通じて対応する保護ポリシーを適用するようになりました。エンタープライズ データへのセキュアでポリシー準拠の検索が可能になり、承認されたドキュメントのみを使う agentic RAG シナリオを実現します。

### Teams の Facilitator が会議中の質問を自動検出・回答

- 原題: Microsoft Teams: Facilitator detects and answers questions
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/26
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558341

Facilitator が Teams 会議中に提起された未回答の質問を自動的に検出し、Web 検索を使って回答を支援するようになりました。参加者が質問を投げかけて誰も答えなかった場合、Facilitator が回答を探す提案を投稿します。参加者が「はい」を選ぶとエージェントが回答を取得します。

### Copilot が文書構造を活用した長いファイルのナビゲーションを改善

- 原題: Microsoft Copilot (Microsoft 365): Copilot improved long files navigation using document structure, enabling more accurate answers and clearer citations.
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/26
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559614

Copilot が長く複雑なファイルを読む方法を改善し、文書構造への理解が深まりました。質問すると Copilot はドキュメント内の最も関連性の高い部分（特定のセクションやページなど）に移動してから回答するため、見落としを減らし、より明確な引用で信頼性が向上します。
