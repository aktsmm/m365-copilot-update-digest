### Microsoft 365 管理センター: 使用状況レポート – Microsoft 365 Copilot コネクタ

- 原題: Microsoft 365 admin center: Usage reports – Microsoft 365 Copilot Connectors
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/05
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=519571

新しい使用状況レポートは、Microsoft 365 Copilot の Copilot コネクタの Microsoft 365 管理センターで利用できるようになります。これは、組織が Microsoft 365 Copilot 内でコネクタがどのように使用されているかを詳細に把握できるように設計されています。このレポートは、コネクタ全体のアクティビティとコネクタによる応答に関する洞察を提供します。

### 強化された Power BI フィルタリングでより深い Copilot の洞察を解き放つ

- 原題: Microsoft Viva: Unlocking Deeper Copilot Insights with Enhanced Power BI Filtering
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/05
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559995

グローバル管理者とインサイト管理者が、予約属性や組織のカスタム属性など、追加の属性を Power BI レポートのフィルターとして有効にできるようにします。 Power BI レポートにアクセスできるユーザーは、これらの新しく有効になったフィルターを使用して、ビジネス ニーズに合わせてレポートを調整できます。

### Copilot ノートブックから Word ドキュメントを作成する

- 原題: Microsoft Copilot (Microsoft 365): Create Word documents from Copilot Notebooks
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/05
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558934

Copilot ノートブックを使用すると、ユーザーはノートブックに収集されたコンテンツと参照を使用して Word ドキュメントを生成できます。 Copilot は、ノートブックのコンテキストに基づいて構造化された文書を作成します。文書は Word で開いて編集でき、ユーザーがアイデアやソース資料から、少ない労力ですぐに使える文書に移行できるようにします。

### Copilot ノートブックでのマルチモーダル キャプチャ (iPhone)

- 原題: OneNote: Multimodal capture in Copilot Notebooks (iPhone)
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/05
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559095

マルチモーダル キャプチャにより、ユーザーは 1 回のセッションで音声の文字起こし、画像の撮影、メモの入力を行うことができます。次に、Copilot は、洞察とユーザーが取得したコンテンツを含む構造化された Copilot ページを作成し、ユーザーが選択した Copilot ノートブックに保存します。対面での会話、ホワイトボードなど、オフラインでの仕事の瞬間をキャプチャするために設計されています。
