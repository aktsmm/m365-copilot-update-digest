### M365 Copilot に Project Manager Agent を追加

- 原題: Microsoft Copilot (Microsoft 365): Project Manager Agent in M365 Copilot
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/31
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=516576

Microsoft 365 の新しいプロジェクト マネージャー エージェントは、作業をより効率的に計画、整理、管理するのに役立ちます。私たちは、プロジェクトの追跡を簡素化するための主要なタスクと計画の管理機能から始めており、今後、追加の高度なプロジェクト管理機能を導入する予定です。

### Word 文書上部から音声概要にアクセス可能に

- 原題: Microsoft Copilot (Microsoft 365): Access audio overviews from the top of your Word document
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/31
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=492615

ドキュメント スナップショットの [概要] タブからドキュメントの概要を音声で生成するように Copilot に依頼します。機能の現在の状態を正確に反映するためにステータスを更新しました。

### Edge の Copilot New Tab Page を刷新

- 原題: Microsoft Edge: Copilot New Tab Page
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/31
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558256

Copilot が提案するアクションや厳選された作業コンテンツを使用して 1 つの検索ボックスから Web を検索、チャット、探索できるため、新しく更新された Copilot の新しいタブ ページでユーザーが作業を迅速に完了できるようになります。ユーザーは、パーソナライズされたニュースやアクティビティを表示することもできます。

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Teams: Enhanced Copilot chat summary
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/31
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=501201

多くのメッセージを見逃していませんか? Copilot はチャット会話内の新しいメッセージの概要を自動的に生成するため、すべてのメッセージを読まなくてもすぐに内容を理解できます。

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Outlook: Copilot provides coaching feedback in chat as you draft, edit and format emails
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/31
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559418

Copilot は、電子メールの下書きに関するコーチング フィードバックをチャット経由で提供し、Outlook のメッセージに提案を直接適用できるようになりました。ユーザーはどの提案を適用するかを選択することができ、Copilot によってメールが適切に更新されます。
