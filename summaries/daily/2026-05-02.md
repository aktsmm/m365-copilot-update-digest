## 全件

### Outlook: Outlook Copilot サイド ウィンドウのアカウント セレクター

- 原題: Outlook: Account Selector in Outlook Copilot Side Pane
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/02
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559991

ユーザーが複数の Copilot 対応アカウントを新しい Outlook に追加している場合、カレンダーの Copilot サイド ウィンドウにアカウント ドロップダウンが表示され、使用している Copilot 対応アカウントが明確になり、Outlook UI のアカウント コンテキストに依存せずにアカウントを切り替えることができます。

### Microsoft Teams: 会議録画の機密ラベルの継承

- 原題: Microsoft Teams: Sensitivity label inheritance for meeting recordings
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/02
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557178

会議の録画は、会議の機密ラベルを自動的に継承するようになりました。機密ラベル ポリシーでラベルの継承が有効になっている場合、ラベル付き会議は同じラベルを MP4 録音に適用し、アクセス制御、データ処理ルール、トランスクリプトに基づく Copilot またはエージェントの応答が一貫して保証されるようにします。

### Microsoft Teams: AI を活用した対面でのメモ
Android 上の Teams Rooms でのファシリテーターとの会議

- 原題: Microsoft Teams: AI-powered notes for in-person
meetings with Facilitator in Teams Rooms on Android
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/02
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560700

ファシリテーター エージェントは、スケジュールされた会議またはハイブリッド会議中にリアルタイムのメモや会議結果をキャプチャするだけでなく、Android 上の Teams Rooms での対面会議のメモ、意思決定、アクションを支援します。ルームコンソールをワンタップするだけでファシリテーターを招待できます。会議のメモが部屋の正面に表示されます...

### Microsoft Copilot (Microsoft 365): 入力ボックスのコンテンツの簡素化

- 原題: Microsoft Copilot (Microsoft 365): Simplifying Content in the Input Box
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/02
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=552592

ユーザーがプラス メニューの [作業コンテンツの追加] を使用してプロンプトにコンテンツを挿入すると、そのコンテンツは入力ボックス UI にピルとしてのみ表示され、ユーザーのプロンプトの一部として繰り返されません。これは、入力ボックスを簡素化し、プロンプト作成用に予約するのに役立ちます。 2026 年 4 月 22 日更新: このアイテムは移動されます...

### Microsoft Teams: インタプリタでの逐次通訳

- 原題: Microsoft Teams: Consecutive interpretation in Interpreter
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/02
- リリース段階: Rolling out
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557180

逐次通訳により、会議参加者全員が順番に通訳することができます。各話者の話が終わった後に通訳が行われ、2 つの言語にわたる対話型のディスカッションをサポートするターンベースのフローが作成されます。
