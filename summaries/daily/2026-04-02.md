### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Copilot (Microsoft 365): [Copilot Search] [Copilot Search] Copilot Chat available when using Copilot Search
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/02
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=537281

Microsoft 365 Copilot ユーザーは、Microsoft 365 Copilot アプリで Copilot Search を使用するときに、右側のサイド ウィンドウから Copilot チャットを利用できるようになります。これにより、ユーザーはページを切り替えたりコンテキストを失ったりすることなく、ユニバーサル検索を閲覧しながら Copilot を利用できるようになります。

### Copilot Notebooks から Excel 作成に対応

- 原題: Microsoft Copilot (Microsoft 365): Create Excel spreadsheets from Copilot Notebooks
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/02
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559480

Copilot Notebooks を使用すると、ユーザーはノートブックに収集されたコンテンツと参照を使用して Excel スプレッドシートを生成できます。 Copilot は、ノートブックのコンテキストに基づいた構造化されたスプレッドシートを作成します。これは Excel で開いて編集でき、ユーザーがアイデアやソース資料からすぐに使用できるスプレッドシートに移行できるように支援します。
