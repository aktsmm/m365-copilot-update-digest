## 全件

### Excel: Copilot を使用してテキスト列を要約する

- 原題: Excel: Summarize text columns with Copilot
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/16
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560531

Copilot はテキスト列を分析して、関連するカテゴリまたはタグを生成できます。たとえば、Copilot は、自由回答形式の質問からアンケート回答の主要なテーマを見つけます。

### Copilot Analytics - Copilot ダッシュボードでの特定可能なエクスポート メトリクス

- 原題: Microsoft Viva: Copilot Analytics - Identifiable Export metrics in the Copilot Dashboard
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/16
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=562648

これにより、ユーザーは、レポート作成やより詳細な分析のために、識別可能な行レベルの Copilot メトリクスを Copilot ダッシュボードからエクスポートできるようになります。この機能は、テナントの管理者がオンに調整できます。

### 会議録画の機密ラベルの継承

- 原題: Microsoft Teams: Sensitivity label inheritance for meeting recordings
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/16
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557178

会議の録画は、会議の機密ラベルを自動的に継承するようになりました。機密ラベル ポリシーでラベルの継承が有効になっている場合、ラベル付き会議は同じラベルを MP4 録音に適用し、アクセス制御、データ処理ルール、トランスクリプトに基づくCopilotまたはエージェントの応答が一貫して保証されるようにします。

### People Skills - 削除と削除の管理制御

- 原題: Microsoft Copilot (Microsoft 365): People Skills- removal and deletion admin control
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/16
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=565905

People Skills 設定の新しい管理コントロールを使用すると、管理者は組織の People Skills データをテナントから完全に削除できます。この新しいコントロールを使用すると、組織はスキル ライブラリ全体と、確認、推測、インポートされたすべてのユーザーのスキル データを削除でき、...
