## 全件

### Copilot ノートブックから Excel スプレッドシートを作成

- 原題: Microsoft Copilot (Microsoft 365): Create Excel spreadsheets from Copilot Notebooks
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/20
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559480

Copilot ノートブックを使用すると、ユーザーはノートブックに収集されたコンテンツと参照を使用して Excel スプレッドシートを生成できます。 Copilot は、ノートブックのコンテキストに基づいた構造化されたスプレッドシートを作成します。これは Excel で開いて編集でき、ユーザーがアイデアやソース資料からすぐに使用できるスプレッドシートに移行できるように支援します。

### インサイト - 新しい Copilot メトリクスが利用可能

- 原題: Microsoft Viva: Insights - New Copilot Metrics Available
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/20
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557981

Microsoft 365 Copilot アプリ、Edge、OneNote で実行されたアクションを含む一連の新しい Copilot 指標と、提案された返信、翻訳、コーチング、クリーン データなどを含む Outlook、Word、Excel、PowerPoint にわたる主要なユース ケースを提供します。これにより、お客様は主要な Copilot についてより深く理解できるようになります。

### Copilot Analytics - Cowork と WorkIQ の支出と使用状況に関する洞察

- 原題: Microsoft Viva: Copilot Analytics - Insights in Cowork and WorkIQ spend and usage
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/20
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=566302

グループまたはチーム レベルでの AI 支出 (Copilot クレジットの使用量) に関する洞察は、ダッシュボードとしての Viva Insights および Advanced Insights で利用できるようになります。このリリースの対象となるサービスは、Cowork および Work IQ API です。ダッシュボード エクスペリエンスは、少なくとも 5 人の直属の部下 (範囲限定アクセス) を持つマネージャーが利用できます。
