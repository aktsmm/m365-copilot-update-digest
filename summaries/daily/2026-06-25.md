## 全件

### Copilot ノートブック から Word 文書生成に対応

- 原題: Microsoft Copilot (Microsoft 365): Create Word documents from Copilot Notebooks
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/25
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558934

Copilot ノートブックに収集したコンテンツや参照情報を使って Word 文書を生成できます。Copilot がノートブックのコンテキストに基づいた構造化ドキュメントを下書きし、Word で開いて編集できます。アイデアや素材から完成度の高い文書を効率的に作成できます。

### Microsoft Teams の会議・チャット機能を更新

- 原題: Microsoft Teams: Facilitator detects and answers questions
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/25
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558341

Teams 会議中に参加者から出た未回答の質問を Facilitator が自動検出し、Web 検索で回答を提案するようになりました。参加者が質問しても誰も答えない場合、Facilitator が回答を検索するか確認します。「はい」を選択するとエージェントが回答を返します。

### Microsoft Purview の会議・チャット機能を更新

- 原題: Microsoft Purview: Data Loss Prevention to restrict processing external emails in Microsoft 365 Copilot and Copilot Chat
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/25
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=561552

Microsoft Purview の DLP を Microsoft 365 Copilot と Copilot Chat に拡張し、外部メールに起因するリスクに対処します。組織外の送信者からのメールを Copilot が処理しないようリアルタイムで制御し、データ漏えいのリスクを低減します。

### Microsoft Purview のセキュリティ・管理・分析機能を強化

- 原題: Microsoft Purview: Data Security Triage Agent in Data Loss Prevention - Reasoning Trace and Confidence Score for agent-triaged alerts
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/06/25
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560597

DLP の Data Security Triage Agent に信頼スコアと推論トレースの可視化機能を導入します。エージェントの判断根拠や確信度が不透明なため手動対応を強いられていた課題に対応し、アナリストがエージェントの判断を信頼して活用できる環境を整えます。
