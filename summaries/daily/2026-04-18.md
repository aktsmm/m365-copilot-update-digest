## 全件

### Microsoft Copilot (Microsoft 365): ビジュアルでよりスマートになりました: Copilot Chat は埋め込み画像を活用して、より豊富で正確な回答を提供し...

- 原題: Microsoft Copilot (Microsoft 365): Now smarter with visuals: Copilot Chat leverages embedded images for richer, more accurate answers.
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/18
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560540

Copilot は、Word 文書 (.docx)、PowerPoint プレゼンテーション (.pptx)、PDF などのファイルに埋め込まれた画像を使用して応答を解釈し、グラウンディングするように強化されました。この機能により、Copilot はチャート、図、スクリーンショットなどの視覚要素から洞察を抽出できるようになり、答えがテキストだけではないことが保証されます。

### Outlook: Copilot は Outlook の検索結果に簡潔な概要を提供するようになりました

- 原題: Outlook: Copilot now provides concise summaries with Outlook search results
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/18
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559989

Outlook の検索結果の上部に Copilot の簡潔な AI 概要が表示されるため、クエリに対する回答がすぐに得られます。ユーザーは、Copilot 概要カードから直接 Copilot チャットを開くことで、より深く掘り下げることができます。

### Microsoft Viva: Viva Glint - Glint アンケートの回答で Copilot がコメントを支援

- 原題: Microsoft Viva: Viva Glint - Copilot assisted commenting in Glint survey taking
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/18
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560595

Viva Glint のアンケート回答者は、Glint のアンケート回答エクスペリエンスで Microsoft Copilot の合理化されたバージョンを使用してコメントを書き直すことができるため、アンケート回答者は自分が提供するフィードバックに対してより自信を持つことができます。識別可能な書き込みパターンを減らすことで、知覚されるセキュリティを向上させることで...

### Microsoft 365 アプリ: Microsoft Loop - ユーザー所有のループ ワークスペースの離脱ユーザー コンテンツ ワークフロー

- 原題: Microsoft 365 app: Microsoft Loop - Departed user content workflows for user-owned Loop workspaces
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/18
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=421612

ユーザーが組織を退職したときに、Copilot Pages などのユーザー所有の Loop ワークスペースの保持および削除のワークフローを管理します。削除ポリシーが有効になる前に、他の従業員または管理者に通知して、重要なデータを保存するための一時的なアクセスを提供します。

### Microsoft Purview: Insider Risk Management - Generative AI アプリ指標用の AI アプリの選択

- 原題: Microsoft Purview: Insider Risk Management-AI app selection for Generative AI apps indicators
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/18
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559992

お客様は、生成 AI アプリの指標を検出するためにどの AI アプリを使用するかを正確に選択できます。これは、「Copilot での危険なプロンプトの入力」、「Copilot からの機密の応答の受信」、「エンタープライズ AI アプリでの危険なプロンプトの入力」、および「機密の応答の受信」の指標に適用されます。
