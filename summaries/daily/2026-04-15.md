## 全件

### Excel: Excel の Copilot の計画モード

- 原題: Excel: Plan Mode for Copilot in Excel
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/15
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560338

Excel の Copilot の計画モードを使用すると、ブック内の内容が更新される前に明確な段階的なアプローチの概要が示されるため、ユーザーは自信を持って複雑な変更を行うことができます。ユーザーは、Copilot がどのようにタスクを完了する予定であるかを確認し、どのデータまたは機能が使用されるかを確認し、必要に応じて計画を調整して、編集内容が適切であることを確認できます。

### Outlook: iOS 用ファイル プレビューアーの Copilot

- 原題: Outlook: Copilot in File Previewer for iOS
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/15
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560340

Outlook でファイルをプレビューするときに、Copilot に直接アクセスします。 [Copilot] ボタンをタップすると、プレビューアを終了せずにドキュメントに関するスマートな提案、概要、回答が得られます。必要なときに添付ファイルやファイルを理解するための迅速なヘルプを入手できます。

### Outlook: Outlook Copilot サイド ウィンドウのアカウント セレクター

- 原題: Outlook: Account Selector in Outlook Copilot Side Pane
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/15
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=559991

ユーザーが複数の Copilot 対応アカウントを新しい Outlook に追加している場合、カレンダーの Copilot サイド ウィンドウにアカウント ドロップダウンが表示され、使用している Copilot 対応アカウントが明確になり、Outlook UI のアカウント コンテキストに依存せずにアカウントを切り替えることができます。

### Microsoft Teams: 要約で会議で生成されたコンテンツを削除する

- 原題: Microsoft Teams: Delete meeting generated content in recap
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/15
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557170

会議の主催者は、要約ページの新しい削除ボタンを使用して、録音、トランスクリプト、AI サマリー、メモなどの会議で生成されたコンテンツを簡単に削除できるようになりました。ミーティング内の共有ファイルは削除されません。

### Microsoft Edge: シャドウ AI

- 原題: Microsoft Edge: Shadow AI
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/15
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560341

現在、管理者は Purview DLP ポリシーを設定して、機密データが Generative AI アプリに送信されないように保護できますが、これがトリガーされてワークフローが停止すると、ユーザーは LLM へのプロンプトの送信がブロックされます。この機能への新しいアドオンは、ユーザーに M365 にリダイレクトされるオプションを提供する新しい UI を表示します。

### Microsoft Teams: Teams 電話エージェント

- 原題: Microsoft Teams: Teams Phone Agent

- リリース段階: In development
- 役割タグ: 作成者向け

### Microsoft Purview: エンドポイントのデータ損失防止 - カスタム ポリシーを通じて、スナップショットのリコールに対する保護を Copilot + PC デバイスに拡張します

- 原題: Microsoft Purview: Endpoint Data Loss Prevention - Expand protection to Copilot + PC devices for Recall snapshots through custom policies
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/15
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=560396

Microsoft Purview Endpoint データ損失防止は、最初はリコール スナップショットをサポートし、制限された機密ラベルと機密情報の種類 (SIT) を含むウィンドウのキャプチャを防止するポリシーが存在するかどうかを判断するために、対象範囲を Copilot + PC に拡大しています。 Purview 管理者は、Endpoint DLP カスタムを作成します。
