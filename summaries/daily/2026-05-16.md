## 全件

### Microsoft Copilot Studio: エージェント ワークフローで MCP 準拠のツールを使用する

- 原題: Microsoft Copilot Studio: Use MCP-compliant tools in agent workflows
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/16
- リリース段階: In development
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=562221

MCP サポートにより、独自のシステム、動的なナレッジ ソース、カスタム アクションをエージェント ワークフローに統合し、ワークフローの信頼性を維持しながら拡張性を加速します。この統合により、エージェントやワークフロー全体での再利用が向上し、統合の重複が削減され、新たなオープン標準の採用を後押しします。

### Microsoft Copilot Studio: コンピューターの使用でワークフローを強化します

- 原題: Microsoft Copilot Studio: Supercharge your workflows with computer use
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/16
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=562220

コンピューター使用ツールを使用して、回復力のある AI を活用した UI 自動化をワークフローに直接追加します。作成者は、Web サイトやアプリの変更に動的に適応する再利用可能な目標指向の自動化を構築できる一方、管理者はアクセス制御、ホストされたブラウザーの管理、環境レベルの権限を通じてガバナンスを維持できます。
