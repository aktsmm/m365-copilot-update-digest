## 全件

### Copilot で scatter image effect に対応

- 原題: Microsoft Copilot (Microsoft 365): Scatter image effect in Microsoft 365 Copilot
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/20
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=537288

Copilot チャットと作成エクスペリエンスで散布イメージ効果を直接使用して、Copilot で生成されたビジュアル アーティファクトをさらにカスタマイズします。

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Copilot (Microsoft 365): Infinite Scroll Chat History
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/20
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557348

無限スクロールにより、ユーザーはスクロール中に追加の会話を自動的に読み込むことで、古いチャット セッションを継続的に閲覧できます。これにより、よりスムーズで最新のチャット履歴エクスペリエンスが作成されます。GA 日: 2026 年 4 月

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Copilot (Microsoft 365): Image Upload in Copilot Chat for Government Cloud
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/20
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=551194

Microsoft 365 Copilot の画像アップロード機能を使用すると、ユーザーは画像をアップロードし、それらの画像に基づいて Copilot から洞察を得ることができます。私たちはこの機能をガバメント クラウド環境に導入し、既存の Copilot エクスペリエンスと同等の機能を提供しながら、該当するガバメント クラウド環境内で機能が動作することを保証します。

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Microsoft Teams: Sensitivity label inheritance for meeting recordings
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/20
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557178

会議の録画は、会議の機密ラベルを自動的に継承するようになりました。機密ラベル ポリシーでラベルの継承が有効になっている場合、ラベル付き会議は同じラベルを MP4 録音に適用し、アクセス制御、データ処理ルール、トランスクリプトに基づくCopilotまたはエージェントの応答が一貫して保証されるようにします。
