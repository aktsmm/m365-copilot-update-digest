- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/22
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=548671

Microsoft は、機密データを含む Web 検索を保護するために、Microsoft 365 Copilot および Copilot Chat 用の Microsoft Purview DLP を拡張しています。このリアルタイム制御は、Microsoft 365 Copilot とエージェントが外部の Web 検索に機密データを使用することを防ぎ、組織がデータ漏洩と過剰共有のリスクを軽減するのに役立ちます。
