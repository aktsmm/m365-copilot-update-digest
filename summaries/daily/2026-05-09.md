## 全件

### Microsoft Copilot (Microsoft 365): アプリとエージェントのインベントリと詳細用の Microsoft Graph API。

- 原題: Microsoft Copilot (Microsoft 365): Microsoft Graph APIs for App &amp; Agent Inventory and Details.
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/09
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=502875

Microsoft 365 では、IT 管理者と開発者が Copilot、Teams、Outlook、その他の MetaOS ホスト全体でアプリとエージェントの管理に確実にプログラムでアクセスできるように設計された新しい Graph API エンドポイントが導入されます。主な機能: • インベントリ API: 社内のすべてのアプリとエージェントの包括的なインベントリを取得します。

### Microsoft Teams: Teams イベントの空室状況シグナル

- 原題: Microsoft Teams: Room availability signal for Teams events
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/05/09
- リリース段階: In development
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=561647

Teams のイベント アプリで、Teams のイベント主催者は、イベント用に選択した部屋またはスペースが指定された時間に利用可能かどうかを確認できるようになります。この機能は、Teams Enterprise ライセンスを持つすべての Teams イベント主催者が利用できます。
