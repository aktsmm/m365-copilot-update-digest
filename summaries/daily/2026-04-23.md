## 全件

### Outlook: Outlook の Copilot チャットを使用してメールを要約する

- 原題: Outlook: Summarize email with Copilot chat in Outlook
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/23
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=503316

Copilot チャットを使用して、メールと添付ファイルを簡単に要約します。これは、Microsoft 365 Copilot ライセンスを持ち、ピン留めが有効になっている Android および iOS ユーザーが利用できます。 2026 年 4 月 21 日更新: 現時点では、デスクトップ、Mac、Web、Windows 向けクラシック Outlook と新しい Outlook の両方へのこの機能のロールアウトをキャンセルしました。

### Microsoft Copilot (Microsoft 365): Unified Plus メニュー (+)

- 原題: Microsoft Copilot (Microsoft 365): Unified Plus menu (+)
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/23
- リリース段階: Launched
- 役割タグ: 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=552596

さまざまなユーザー アクションをプラス メニュー (+) にまとめることで、Copilot Chat の入力ボックスを簡素化しています。統合されたプラス メニュー (+) により、ユーザーは引き続きプロンプトにコンテンツを追加できると同時に、ツール/エージェントにアクセスし、Copilot を接地するために使用されるデータ ソースを管理できるようになります。

### Microsoft Viva: Glint: Copilot ハイライトを使用した AI 生成の調査分析情報

- 原題: Microsoft Viva: Glint: AI-generated survey insights with Copilot Highlights
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/23
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=558111

Viva Glint の Copilot は、従業員アンケート結果の AI を活用した概要をチーム サマリー レポートとエグゼクティブ サマリー レポート内で直接生成できるようになり、リーダーがチーム全体の強み、機会、傾向を迅速に理解できるようになりました。 Copilot Highlights は、注目すべきスコアの変化、ベンチマークなど、重要な洞察を明らかにします。

### Microsoft Edge: Google の仕事検索バナー

- 原題: Microsoft Edge: Work Search Banner for Google
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/23
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
