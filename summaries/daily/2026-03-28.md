### Edge アドレスバーで要約提案を表示

- 原題: Microsoft Edge: v.144 - Contextual nudges on address bar offering help summarizing webpages.
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/28
- リリース段階: Launched
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=489822

Microsoft Edge for Business では、ユーザーが Microsoft 365 Copilot Chat を使用して、開いているページの概要をアドレス バーから簡単に取得できる方法を導入する予定です。この機能は、ユーザーが関連コンテンツをより簡単に理解し、時間を節約できるようにすることを目的としています。

### Copilot Dashboard に満足度メトリクスを追加

- 原題: Microsoft Viva: Satisfaction Rate Metric for Microsoft 365 Copilot in Copilot Dashboard
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/03/28
- リリース段階: Launched
- 役割タグ: 管理者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=496655

Microsoft 365 Copilot に対するユーザーのセンチメントを追跡します。 Copilot インタラクション後の賛成評価と反対評価の内訳を分析することで、ユーザーが Copilot の価値をどのように認識しているかを理解します。長期にわたる傾向とグループごとの内訳を確認します。

- リリース段階: In development
- 役割タグ: 管理者向け

- 公開日: 2026/03/28
- リリース段階: Rolling out
- 役割タグ: 管理者向け / 作成者向け

Researcher は、Microsoft 365 Copilot アドオン ライセンスを持つ Microsoft 365 ビジネスおよびエンタープライズ ユーザーが利用できます。 Researcher は、Microsoft 365 Copilot の高度な深層推論エージェントであり、深層リサーチ アシスタントとして機能することで、複雑な複数ステップのタスクに取り組むように設計されています。
