### Work IQ API パブリック プレビュー: A2A を使用して Copilot を利用したエージェントを構築する

- 原題: Work IQ API public preview: Build Copilot powered agents with A2A
- ソース: Copilot Studio Blog
- 公開日: 2026/05/12
- リリース段階: GA
- 役割タグ: 作成者向け
- URL: https://techcommunity.microsoft.com/t5/copilot-studio-blog/work-iq-api-public-preview-build-copilot-powered-agents-with-a2a/ba-p/4516286

現在、Work IQ API がパブリック プレビューで利用可能になり、開発者やパートナー向けに Microsoft 365 Copilot で使用されているものと同じ基盤となるインテリジェンス機能へのアクセスが拡張されます。
Work IQ では、開発者がプログラムから Copilot にアクセスできるため、Copilot の機能を使用するエージェントやアプリを構築できます。
