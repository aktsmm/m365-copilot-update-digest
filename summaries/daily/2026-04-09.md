### Agent Builder から Agent Store へ申請可能に

- 原題: Microsoft Copilot (Microsoft 365): Submit agent to Agent Store from Agent Builder
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/09
- リリース段階: In development
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=557173

お客様は、M365 管理センターで管理者がレビューおよび承認した後、Agent Builder で構築されたカスタム Copilot エージェントを「組織によって構築」セクションで Agent Store に送信できます。この管理されたフローにより、管理者は送信されたエージェントを確認、承認、公開できるため、他のユーザーがエージェントを検出して使用できるようになります。

- 公開日: 2026/04/09
- リリース段階: In development

### Microsoft 365 Copilot の会議・チャット機能を更新

- 原題: Outlook: Copilot suggests how you should RSVP to meeting requests
- ソース: Microsoft 365 Roadmap
- 公開日: 2026/04/09
- リリース段階: Update
- 役割タグ: 管理者向け / 作成者向け
- URL: https://www.microsoft.com/microsoft-365/roadmap?id=487412

Copilot は、会議の重要性、空き状況、勤務時間に基づいて、ユーザーが会議出席依頼にどのように応答するかを提案します。 Copilot の支援により、ユーザーは会議出席依頼の優先順位付けを効率化し、優先順位付けされる会議の数を増やし、優先順位付けに必要な時間を短縮できます。この機能は現時点ではリリースされません。
